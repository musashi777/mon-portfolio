Guide Détaillé - Installation Serveur GLPI

📋 Aperçu du Projet

Technologies :
  • GLPI 10.0.15
  • Debian 12
  • Apache 2.4
  • MariaDB 10.11
  • PHP 8.2
Durée :

2-3 heures

Niveau :

Intermédiaire

Résultats :
  • ✅ Serveur fonctionnel
  • ✅ Interface web
  • ✅ Base sécurisée

🎯 Objectif du Projet
#

L’objectif de ce projet est de documenter la procédure complète pour installer et configurer un serveur GLPI, un outil open-source de gestion des services informatiques (ITSM) et de gestion des actifs (ITAM). Cette solution permet de gérer efficacement un parc informatique, le suivi des tickets d’incidents et l’inventaire matériel et logiciel.

Configuration Détaillée - VPN Site-à-Site

🌐 Aperçu du Projet VPN

Technologies :
  • IPsec (IKEv2)
  • Pfsense 2.7+
  • AES-256-GCM
  • SHA-256
  • DH Group 14
Configuration :

Site-à-Site

Chiffrement :

Fort (AES-256)

Résultats :
  • ✅ Tunnel stable
  • ✅ Sécurisé
  • ✅ Performant

🎯 Contexte du Projet
#

L’objectif de ce projet était de relier deux réseaux locaux distants de manière sécurisée à travers Internet en utilisant des pare-feux Pfsense. Cette configuration permet aux deux sites de communiquer comme s’ils étaient sur le même réseau local, tout en garantissant la confidentialité et l’intégrité des données.
